Sonde DHT11

API Sonde_DHT11


Le capteur de température et d'humidité est branché sur un nodeMcu.
Un relevé est transmis à la base de données du serveur OVH toutes les 10 minutes avec la requête POST.

Documentation sur les requêtes disponibles de l'API


L'url de base est la suivante : /vps330078.ovh.net/sonde/api
Il faut ensuite ajouter la suite en fonction de la requête choisie.


/meas/day/last

Méthode : GET

Objet : Récupérer le dernier couple de valeurs enregistré dans la base de données

Url complete : /vps330078.ovh.net/sonde/api/meas/day/last

Paramètres : aucun

Réponse : Json {'date': 'valeur', 'temp': 'valeur', 'humd': 'valeur'}

/meas/day/current

Méthode : GET

Objet : Récupérer les couples de valeurs du jour enregistrés dans la base de données

Url complete : /vps330078.ovh.net/sonde/api/meas/day/current

Paramètres : aucun

Réponse : Json {['date': 'valeur', 'temp': 'valeur', 'humd': 'valeur'],...}

/meas/day/{date}

Méthode : GET

Objet : Récupérer les couples de valeurs d'un jour particulier enregistrés dans la base de données

Url complete : /vps330078.ovh.net/sonde/api/meas/day/{date}

Paramètres : date (2017-03-09)

Réponse : Json {['date': 'valeur', 'temp': 'valeur', 'humd': 'valeur'],...}

/meas/days/{date1}/{date2}

Méthode : GET

Objet : Récupérer, entre deux jours particuliers, les couples de valeurs enregistrés dans la base de données

Url complete : /vps330078.ovh.net/sonde/api/meas/days/{date1}/{date2}

Paramètres : dates (2017-03-09)

Réponse : Json {['date': 'valeur', 'temp': 'valeur', 'humd': 'valeur'],...}

/meas/month/current

Méthode : GET

Objet : Récupérer les couples de valeurs du mois courant enregistrés dans la base de données

Url complete : /vps330078.ovh.net/sonde/api/meas/month/current

Paramètres : aucun

Réponse : Json {['date': 'valeur', 'temp': 'valeur', 'humd': 'valeur'],...}

/meas/month/{month1}

Méthode : GET

Objet : Récupérer les couples de valeurs d'un mois particulier enregistrés dans la base de données

Url complete : /vps330078.ovh.net/sonde/api/meas/month/{month1}

Paramètres : mois (2017-03)

Réponse : Json {['date': 'valeur', 'temp': 'valeur', 'humd': 'valeur'],...}

/meas/months/{month1}/{month2}

Méthode : GET

Objet : Récupérer, entre deux mois particuliers, les couples de valeurs enregistrés dans la base de données

Url complete : /vps330078.ovh.net/sonde/api/meas/month/{month1}/{month2}

Paramètres : mois (2017-03)

Réponse : Json {['date': 'valeur', 'temp': 'valeur', 'humd': 'valeur'],...}

/meas/new

Méthode : POST

Objet : Enregistrer le couple de valeurs mesuré par le nodeMCU dans la base de données

Url complete : /vps330078.ovh.net/sonde/api/meas/new

Paramètres : json {'value1': 'valeur', 'value2': 'valeur'}

Réponse : format html : "New measurement !!!=> température : 'valeur'; humidité : 'valeur'"
