Cette application Web a été développée pour le projet #9 du cycle de formation de la DTA 2017.

Le project consiste à mettre à disposition des données de température et d'humidité pour des applications tièrces grace à une API. Ces données sont mesurées à l'aide d'un NodeMcu dans la maison, puis envoyées à un serveur avec une requête POST de l'API.
Ces relevés sont accessibles grâce aux requêtes GET de l'API, qui sont utilisées dans cette application.
